# Telegram bot game

Clone of the Monopoly board game in a Telegram bot hosted in the cloud

## Build

Compile with `cargo` for Windows (for local testing) and Linux (for the cloud)
Then run `build.ps1`

```bash
cd monopoly
cargo build --release
cd ..
```
```powershell
cd monopoly
cargo build --release
cd ..
.\build.ps1
```
